EatPan

Decentralized P2P chat ecosystem. No servers, no censorship, no limits. Built on libp2p with DHT discovery, offline caching, and auto-updates.

⬇ Download Client ⭐ GitHub

Features

🔗

DHT Discovery Chain

Automatic peer discovery: L2 → L1 → AWS DHT fallback. No single point of failure.

📡

GossipSub Messaging

Real-time chat via GossipSub protocol. Topics for rooms, DMs, and groups.

💾

Offline Caching

Messages cached locally when offline. Auto-sync to L2 when reconnected.

⚡

L3 Seeder Mode

Any peer can upgrade to L3 Seeder — buffer messages and relay to the network.

🏠

mDNS Local Discovery

Find peers on your LAN automatically. Zero configuration required.

🔄

Auto-Updates

Seamlessly update to the latest version via GitHub Releases.

Architecture

L4

Peer Client

End-user chat application. Connects to L3/L2/L1 via DHT. Caches messages offline.

↕
L3

Seeder

L4 client with seeder mode enabled. Buffers and relays messages, announces in DHT.

↕
L2

SuperNode

Full DHT server with leader election. Ingests batches from L4, circuit relay for NAT.

↕
L1

Admin (God Mode)

Global oversight. Remote control, ban/annihilate, Supabase cloud backup.
